# FPGA-Traffic-Light-Controller
# FPGA Traffic Light Controller with Emergency Vehicle Priority

## Overview
This project implements a Traffic Light Controller using Verilog HDL on FPGA.

## Features
- Finite State Machine (FSM) based design
- North-South and East-West traffic control
- Emergency vehicle priority override
- Behavioral simulation using Vivado
- FPGA implementation ready

## Tools Used
- Verilog HDL
- Xilinx Vivado
- Xilinx Spartan FPGA

## Project Structure
- src/ : Verilog design files
- testbench/ : Simulation testbench
- simulation/ : Waveform screenshots
- rtl/ : RTL schematic
- docs/ : FSM diagrams and documentation
